# Downloads

Coloque aqui o arquivo real que será baixado pelo público.

Exemplo:

```text
cloudflare-pages/downloads/PassadorDeSlides-v0.1.0.zip
```

Depois, ajuste o arquivo:

```text
cloudflare-pages/data/update.json
```

Campos principais:

- `version`
- `download_url`
- `asset_name`
- `published_at`
- `notes`
